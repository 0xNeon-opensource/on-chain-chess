Start off with a new chess game by following these simple steps:

  • Select the account to use, enter your username and select the start color to start a game
  • Your game is being created in the blockchain. Once your game is ready, it will appear as "Open game" in the navigation
  • Invite friends to join your game by sending the game id to them. You can find the game ID on the games page.
  • Once another player has joined, the game begins. You will also receive a notification about this.
